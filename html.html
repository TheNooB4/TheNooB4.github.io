1. Introduction to HTML

HTML (Hypertext Markup Language) is the standard markup language used to create and structure web pages. It provides the basic building blocks for creating the content and layout of a website.

<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>
    <h1>Welcome to my website!</h1>
    <p>This is a simple HTML page.</p>
</body>
</html>

This basic HTML structure includes the `<html>`, `<head>`, and `<body>` elements, demonstrating the core components of an HTML document.

2. Basic HTML Structure

Every HTML document has a specific structure that includes the following elements:

  • `<!DOCTYPE html>`: Declares the document type as HTML5.
  • `<html>`: Encloses the entire HTML document.
  • `<head>`: Contains metadata and other non-visible information about the page.
  • `<body>`: Holds the visible content of the web page.
<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
    <!-- Page content goes here -->
</body>
</html>

This code demonstrates the basic structure of an HTML document, including the `<head>` and `<body>` sections.

3. Headings and Paragraphs

HTML provides six levels of headings, from `<h1>` (the largest) to `<h6>` (the smallest), which are used to structure the content of a web page.

<h1>This is a top-level heading</h1>
<h2>This is a second-level heading</h2>
<h3>This is a third-level heading</h3>

<p>This is a paragraph of text.</p>
<p>This is another paragraph of text.</p>

Paragraphs are defined using the `<p>` element, and they are used to group related content together.

4. Links and Images

HTML allows you to create hyperlinks using the `<a>` element, and to embed images using the `<img>` element.

<a href="https://www.example.com">Click me to visit Example.com</a>

<img src="example-image.jpg" alt="Example Image">

The `href` attribute in the `<a>` element specifies the URL of the link, and the `src` attribute in the `<img>` element specifies the path to the image file. The `alt` attribute provides alternative text for the image, which is displayed if the image cannot be loaded.

5. Lists

HTML supports two types of lists: ordered lists (`<ol>`) and unordered lists (`<ul>`).

<h3>Ordered List</h3>
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

<h3>Unordered List</h3>
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ul>

The `<li>` element is used to define each item in the list.

6. Tables

HTML provides the `<table>`, `<tr>`, `<th>`, and `<td>` elements to create tables.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
        <td>London</td>
    </tr>
</table>

The `<th>` element is used for table headers, and the `<td>` element is used for regular table cells.

7. Forms

HTML provides the `<form>` element to create interactive user input forms.

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <label for="message">Message:</label>
    <textarea id="message" name="message"></textarea>

    <button type="submit">Submit</button>
</form>

This form includes input fields for name, email, and a message, as well as a submit button. The `<label>` element is used to provide a label for each input field, and the `type` attribute of the `<input>` element specifies the type of input (e.g., text, email).

8. Semantic Elements

HTML5 introduced a set of semantic elements that provide more meaning to the content of a web page.

<header>
    <h1>My Website</h1>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>
</header>

<main>
    <section>
        <h2>Welcome to my website</h2>
        <p>This is the main content of the page.</p>
    </section>
</main>

<footer>
    <p>Copyright &copy; 2023</p>
</footer>

In this example, we use semantic elements like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` to provide a more meaningful structure to the web page.

9. HTML5 Features

HTML5 introduced several new features and elements that enhance the functionality and accessibility of web pages.

<video src="example-video.mp4" controls>
    Your browser does not support the video tag.
</video>

<audio src="example-audio.mp3" controls>
    Your browser does not support the audio tag.
</audio>

<canvas id="myCanvas">
    Your browser does not support the canvas tag.
</canvas>

<input type="date" id="date-input">

In this example, we demonstrate the use of the `<video>`, `<audio>`, `<canvas>`, and HTML5 input types like `date`.

10. Best Practices

Here are some best practices to keep in mind when writing HTML:

  • Use semantic elements to provide meaning to the content.
  • Maintain proper nesting and indentation of HTML elements.
  • Use meaningful and descriptive names for elements and attributes.
  • Provide alternative text (alt) for images and other media.
  • Validate your HTML code to ensure it's well-formed and follows standards.
  • Keep your HTML code clean and organized for better maintainability.
