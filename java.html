1. Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). It is widely used for a variety of applications, including web development, mobile development, enterprise software, and more.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

This simple "Hello, World!" program demonstrates the basic structure of a Java program, including the `public` access modifier, the `class` keyword, and the `main` method.

2. Variables and Data Types

Java provides a variety of data types, including primitive types (e.g., `int`, `double`, `boolean`) and reference types (e.g., `String`, `Array`, `Class`).

int age = 25;
double pi = 3.14159;
boolean isStudent = true;
String name = "Alice";

In this example, we declare variables of different data types and assign values to them.

3. Operators and Expressions

Java provides a wide range of operators, including arithmetic, assignment, comparison, and logical operators, which can be used to create expressions.

int a = 10, b = 3;
System.out.println("Addition: " + (a + b));
System.out.println("Subtraction: " + (a - b));
System.out.println("Multiplication: " + (a * b));
System.out.println("Division: " + (a / b));
System.out.println("Modulus: " + (a % b));
System.out.println("Equal: " + (a == b));
System.out.println("Not Equal: " + (a != b));

This code demonstrates the use of various operators to perform arithmetic and comparison operations.

4. Control Flow Statements

Java provides several control flow statements, such as `if-else`, `switch`, `for`, `while`, and `do-while`, to control the execution of your code.

int age = 18;
if (age < 18) {
    System.out.println("You are a minor");
} else if (age == 18) {
    System.out.println("You just became an adult");
} else {
    System.out.println("You are an adult");
}

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

These examples demonstrate the use of `if-else` statements and `for` loops in Java.

5. Methods

Methods in Java are reusable blocks of code that perform specific tasks. They can take parameters and return values.

public int add(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    int result = add(5, 3);
    System.out.println(result); // Output: 8

    MyClass obj = new MyClass();
    result = obj.add(10, 4);
    System.out.println(result); // Output: 14
}

In this example, we define an `add` method that takes two integers as parameters and returns their sum. We then call this method from the `main` method to demonstrate its usage.

6. Arrays and Collections

Java provides built-in data structures, such as arrays and collections (e.g., `ArrayList`, `HashMap`), to store and manipulate data.

// Arrays
int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // Output: 1

// ArrayList
ArrayList<String> fruits = new ArrayList<>();
fruits.add("apple");
fruits.add("banana");
fruits.remove(0);
System.out.println(fruits.get(0)); // Output: banana

These examples demonstrate the use of arrays and the `ArrayList` collection in Java.

7. Object-Oriented Programming

Java is an object-oriented programming language, which means it supports the creation of classes and objects.

public class Dog {
    private String name;
    private int age;

    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String bark() {
        return this.name + " says Woof!";
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog("Buddy", 5);
        System.out.println(myDog.bark()); // Output: Buddy says Woof!
    }
}

In this example, we define a `Dog` class with a constructor, private member variables, and a `bark` method. We then create an instance of the `Dog` class and call the `bark` method.

8. Exception Handling

Java provides a structured way to handle unexpected situations or errors that may occur during program execution using exception handling.

public static void main(String[] args) {
    try {
        int result = 10 / 0;
        System.out.println("Result: " + result);
    } catch (ArithmeticException e) {
        System.out.println("Error: " + e.getMessage());
    } catch (Exception e) {
        System.out.println("Unknown error occurred");
    }
}

This example demonstrates the use of a `try-catch` block to handle an `ArithmeticException` that occurs when attempting to divide by zero.

9. File I/O

Java provides classes and methods for reading from and writing to files, allowing you to interact with the file system.

try {
    FileWriter writer = new FileWriter("example.txt");
    writer.write("Hello, World!");
    writer.close();

    FileReader reader = new FileReader("example.txt");
    BufferedReader bufferedReader = new BufferedReader(reader);
    String line;
    while ((line = bufferedReader.readLine()) != null) {
        System.out.println(line);
    }
    reader.close();
} catch (IOException e) {
    System.out.println("An error occurred: " + e.getMessage());
}

This example demonstrates how to write to a file and then read from the same file using Java's file I/O capabilities.

10. Advanced Java Topics

Java has many advanced features and concepts that you can explore further, including:

  • Generics
  • Lambdas and Functional Programming
  • Streams and Parallel Processing
  • Concurrency and Multithreading
  • Java Enterprise Edition (Java EE)
  • Spring Framework
  • Android Development
// Generics
public class Box<T> {
    private T item;

    public void set(T item) {
        this.item = item;
    }

    public T get() {
        return this.item;
    }
}

// Lambdas
Runnable r = () -> {
    System.out.println("Hello from a lambda!");
};
r.run();

// Streams
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
int sum = numbers.stream()
                  .filter(n -> n % 2 == 0)
                  .mapToInt(Integer::intValue)
                  .sum();
System.out.println(sum); // Output: 6

These examples demonstrate the use of generics, lambdas, and streams in Java, which are powerful features for writing concise and expressive code.
