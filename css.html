1. Introduction to CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in a markup language, such as HTML. It allows you to control the layout, color, font, and other visual aspects of web pages.

body {
    font-family: Arial, sans-serif;
    color: #333;
    background-color: #f4f4f4;
}

h1 {
    color: #0070c9;
    font-size: 2.5em;
}

p {
    line-height: 1.6;
}

This CSS code sets the font, color, and background-color for the `body` element, and styles the `h1` and `p` elements.

2. CSS Selectors

CSS selectors are used to target specific HTML elements to apply styles. CSS supports various types of selectors, including:

  • Element selectors (e.g., `h1`, `p`, `div`)
  • Class selectors (e.g., `.my-class`)
  • ID selectors (e.g., `#my-id`)
  • Attribute selectors (e.g., `[type="text"]`)
  • Combinators (e.g., `div p`, `div > p`, `div + p`, `div ~ p`)
/* Element selector */
h1 {
    font-size: 2em;
}

/* Class selector */
.my-class {
    color: #0070c9;
}

/* ID selector */
#my-id {
    font-weight: bold;
}

/* Attribute selector */
[type="text"] {
    border: 1px solid #ccc;
}

/* Combinator */
div p {
    margin-bottom: 10px;
}

These examples demonstrate how to use different types of CSS selectors to target and style HTML elements.

3. CSS Properties

CSS properties are used to define the visual appearance of HTML elements. Some common CSS properties include:

  • Font properties (e.g., `font-family`, `font-size`, `font-weight`)
  • Color properties (e.g., `color`, `background-color`)
  • Box model properties (e.g., `width`, `height`, `margin`, `padding`, `border`)
  • Layout properties (e.g., `display`, `position`, `float`, `flex`)
  • Text properties (e.g., `text-align`, `text-decoration`, `line-height`)
h1 {
    font-family: 'Arial', sans-serif;
    font-size: 2.5em;
    font-weight: bold;
    color: #0070c9;
    margin-bottom: 20px;
}

p {
    color: #333;
    line-height: 1.6;
    text-align: justify;
}

.my-box {
    width: 300px;
    height: 200px;
    background-color: #f4f4f4;
    border: 1px solid #ccc;
    padding: 20px;
    margin: 20px 0;
}

These examples demonstrate how to use various CSS properties to style the appearance of HTML elements.

4. CSS Layout

CSS provides several ways to control the layout of web pages, including:

  • The `display` property (e.g., `block`, `inline`, `flex`, `grid`)
  • The `position` property (e.g., `static`, `relative`, `absolute`, `fixed`)
  • The `float` and `clear` properties
  • Flexbox and Grid layouts
/* Flexbox layout */
.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1;
    margin: 10px;
}

/* Grid layout */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 20px;
}

.grid-item {
    background-color: #f4f4f4;
    padding: 20px;
}

These examples demonstrate the use of Flexbox and Grid layouts to control the positioning and alignment of HTML elements.

5. Responsive Web Design

Responsive web design is the practice of creating web pages that adapt to different screen sizes and devices. CSS provides several tools to achieve responsive design, such as:

  • Media queries
  • Flexible layouts (e.g., Flexbox, Grid)
  • Responsive typography (e.g., `vw`, `vh` units)
  • Responsive images and media
/* Media query for small screens */
@media (max-width: 768px) {
    h1 {
        font-size: 1.8em;
    }

    .container {
        flex-direction: column;
    }

    .item {
        flex-basis: auto;
    }
}

This example demonstrates the use of a media query to apply different styles for small screen sizes, such as adjusting the font size of the `h1` element and changing the layout of the `.container` element.

6. CSS Typography

CSS provides a wide range of properties to control the appearance of text, including:

  • Font properties (e.g., `font-family`, `font-size`, `font-weight`)
  • Text properties (e.g., `color`, `text-align`, `text-decoration`, `line-height`)
  • Web fonts (e.g., `@font-face`)
h1 {
    font-family: 'Roboto', sans-serif;
    font-size: 3em;
    font-weight: 700;
    color: #0070c9;
    line-height: 1.2;
}

p {
    font-family: 'Open Sans', sans-serif;
    font-size: 1.1em;
    color: #333;
    line-height: 1.6;
    text-align: justify;
}

This example demonstrates how to use various CSS properties to control the typography of the `h1` and `p` elements, including font family, size, weight, color, and line height.

7. The CSS Box Model

The CSS box model describes the rectangular box that surrounds every HTML element. It consists of the following components:

  • Content: The actual content of the element (e.g., text, images)
  • Padding: The space between the content and the border
  • Border: The line surrounding the padding and content
  • Margin: The space between the border and the surrounding elements
.box {
    width: 300px;
    height: 200px;
    padding: 20px;
    border: 2px solid #ccc;
    margin: 30px;
}

This example demonstrates how the box model properties (width, height, padding, border, and margin) affect the size and spacing of an HTML element.

8. Color and Backgrounds

CSS provides various ways to set the color and background of HTML elements, including:

  • Color properties (e.g., `color`, `background-color`)
  • Gradient backgrounds (e.g., `linear-gradient`, `radial-gradient`)
  • Background images (e.g., `background-image`, `background-size`, `background-repeat`)
body {
    color: #333;
    background-color: #f4f4f4;
}

h1 {
    color: #0070c9;
    background-color: #e6f3ff;
    padding: 20px;
}

.gradient-bg {
    background: linear-gradient(to right, #0070c9, #00a0d2);
    color: white;
    padding: 50px;
}

.image-bg {
    background-image: url('background.jpg');
    background-size: cover;
    background-repeat: no-repeat;
    padding: 100px;
}

These examples demonstrate how to use CSS to set the color and background of HTML elements, including the use of gradients and background images.

9. Pseudo-classes and Pseudo-elements

CSS pseudo-classes and pseudo-elements allow you to target specific states or parts of an HTML element:

  • Pseudo-classes (e.g., `:hover`, `:active`, `:focus`, `:nth-child()`)
  • Pseudo-elements (e.g., `::before`, `::after`, `::first-line`, `::selection`)
a:link {
    color: #0070c9;
    text-decoration: none;
}

a:visited {
    color: #00a0d2;
}

a:hover {
    text-decoration: underline;
}

button::before {
    content: "âœ¨ ";
}

p::first-line {
    font-weight: bold;
}

These examples demonstrate the use of pseudo-classes to style links in different states, and pseudo-elements to add content before a button and style the first line of a paragraph.

10. Advanced CSS Topics

CSS has many advanced features and techniques that you can explore further, including:

  • CSS Animations and Transitions
  • CSS Transforms
  • CSS Variables (Custom Properties)
  • CSS Frameworks (e.g., Bootstrap, Tailwind CSS)
  • CSS Preprocessors (e.g., Sass, Less)
/* CSS Animations */
@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

.fade-in {
    animation: fadeIn 1s ease-out;
}

/* CSS Variables */
:root {
    --primary-color: #0070c9;
    --secondary-color: #00a0d2;
    --text-color: #333;
}

h1 {
    color: var(--primary-color);
}

p {
    color: var(--text-color);
}

These examples demonstrate the use of CSS animations and CSS variables, which are powerful tools for creating dynamic and customizable web designs.
