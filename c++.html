1. Introduction to C++

C++ is a general-purpose, object-oriented programming language developed by Bjarne Stroustrup in 1979. It is an extension of the C programming language and is widely used for a variety of applications, including system programming, game development, and high-performance computing.

#include 

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

This simple "Hello, World!" program demonstrates the basic structure of a C++ program, including the use of the `iostream` library and the `main` function.

3. Control Flow

Control flow statements in C++ allow you to control the execution of your code based on certain conditions.

If-Else Statements

#include 

int main() {
    int age = 18;

    if (age < 18) {
        std::cout << "You are a minor" << std::endl;
    } else if (age == 18) {
        std::cout << "You just became an adult" << std::endl;
    } else {
        std::cout << "You are an adult" << std::endl;
    }

    return 0;
}

For Loops

#include 

int main() {
    std::string fruits[] = {"apple", "banana", "cherry"};

    for (std::string fruit : fruits) {
        std::cout << "I like " << fruit << std::endl;
    }

    for (int i = 0; i < 5; i++) {
        std::cout << i << std::endl;
    }

    return 0;
}

While Loops

#include 

int main() {
    int count = 0;

    while (count < 5) {
        std::cout << "Count is " << count << std::endl;
        count++;
    }

    return 0;
}

4. Functions

Functions in C++ are reusable blocks of code that perform specific tasks.

#include 

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int main() {
    int result = add(5, 3);
    std::cout << result << std::endl; // 8

    result = subtract(10, 4);
    std::cout << result << std::endl; // 6

    auto square = [](int x) { return x * x; };
    std::cout << square(4) << std::endl; // 16

    return 0;
}

In this example, we define two functions, `add` and `subtract`, that perform the respective operations. We also demonstrate the use of a lambda function to create a `square` function.

6. Object-Oriented Programming

C++ supports the object-oriented programming (OOP) paradigm, which allows you to create and work with classes and objects.

#include 

class Dog {
public:
    Dog(std::string name, int age) {
        this->name = name;
        this->age = age;
    }

    std::string bark() {
        return this->name + " says Woof!";
    }

private:
    std::string name;
    int age;
};

int main() {
    Dog myDog("Buddy", 5);
    std::cout << myDog.bark() << std::endl; // Buddy says Woof!

    class GuideDog : public Dog {
    public:
        GuideDog(std::string name, int age, bool isGuide)
            : Dog(name, age), isGuide(isGuide) {}

        std::string guide() {
            return this->name + " is guiding its owner.";
        }

    private:
        bool isGuide;
    };

    GuideDog guideDog("Bella", 7, true);
    std::cout << guideDog.bark() << std::endl; // Bella says Woof!
    std::cout << guideDog.guide() << std::endl; // Bella is guiding its owner.

    return 0;
}

In this example, we define a `Dog` class with a constructor that initializes the `name` and `age` attributes, and a `bark` method that returns the dog's bark. We then create an instance of the `Dog` class and call the `bark` method.

We also demonstrate inheritance by creating a `GuideDog` class that inherits from the `Dog` class and adds an `isGuide` attribute and a `guide` method.

7. Memory Management

C++ provides manual memory management through pointers and dynamic memory allocation.

#include 

int main() {
    // Pointers
    int x = 10;
    int* p = &x;
    std::cout << *p << std::endl; // 10

    // Dynamic Memory Allocation
    int* arr = new int[5];
    arr[0] = 1;
    arr[1] = 2;
    // ...
    delete[] arr;

    return 0;
}

In this example, we demonstrate the use of pointers to access the value of a variable, as well as dynamic memory allocation and deallocation using `new` and `delete`.

8. File Handling

C++ provides file handling capabilities through the `fstream` library.

#include 
#include 

int main() {
    // Writing to a file
    std::ofstream file("example.txt");
    file << "Hello, World!" << std::endl;
    file.close();

    // Reading from a file
    std::ifstream inFile("example.txt");
    std::string line;
    while (std::getline(inFile, line)) {
        std::cout << line << std::endl;
    }
    inFile.close();

    // Appending to a file
    std::ofstream appendFile("example.txt", std::ios::app);
    appendFile << "This is a new line." << std::endl;
    appendFile.close();

    return 0;
}

In this example, we use the `ofstream` class to write to a file, the `ifstream` class to read from a file, and the `ios::app` mode to append data to a file.

9. Exception Handling

C++ provides a way to handle unexpected situations or errors that may occur during program execution.

#include 

int main() {
    try {
        int result = 10 / 0;
        std::cout << "Result: " << result << std::endl;
    } catch (const std::exception& e) {
        std::cerr << "Error: " << e.what() << std::endl;
    } catch (...) {
        std::cerr << "Unknown error occurred" << std::endl;
    }

    return 0;
}

In this example, we attempt to divide 10 by 0, which will raise a `std::exception`. We catch the exception using the `try-catch` block and print an error message. The `catch (...)` block is used to handle any other types of exceptions that may occur.

10. Advanced Topics

C++ has many advanced features and concepts that you can explore further.

Templates

Templates allow you to write generic code that can work with different data types.

#include 

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    std::cout << add<int>(5, 3) << std::endl; // 8
    std::cout << add<double>(3.14, 2.71) << std::endl; // 5.85
    return 0;
}

Lambdas

Lambdas are anonymous functions that can be used as function arguments or stored in variables.

#include 
#include 
#include 

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    // Using a lambda function
    std::for_each(numbers.begin(), numbers.end(), [](int num) {
        std::cout << num << " ";
    });
    std::cout << std::endl;

    // Storing a lambda in a variable
    auto square = [](int x) { return x * x; };
    std::cout << square(4) << std::endl; // 16

    return 0;
}

Smart Pointers

Smart pointers in C++ provide automatic memory management, reducing the risk of memory leaks.

#include 
#include 

class MyClass {
public:
    MyClass() { std::cout << "MyClass created" << std::endl; }
    ~MyClass() { std::cout << "MyClass destroyed" << std::endl; }
};

int main() {
    std::unique_ptr<MyClass> ptr1(new MyClass());
    std::shared_ptr<MyClass> ptr2(new MyClass());

    // ptr1 and ptr2 will be automatically destroyed when they go out of scope
    return 0;
}
