1. Introduction to Bash

Bash (Bourne-Again SHell) is a Unix shell and command language that is widely used for scripting and automating tasks on Unix-based systems, including Linux and macOS.

#!/bin/bash
echo "Hello, World!"

This simple "Hello, World!" Bash script demonstrates the basic structure of a Bash script, including the shebang line and the `echo` command.

2. Bash Variables and Data Types

Bash supports several data types, including strings, integers, and arrays. Variables are used to store and manipulate data in Bash scripts.

name="Alice"
age=25
is_student=true
fruits=("apple" "banana" "orange")

echo "Name: $name"
echo "Age: $age"
echo "Is student? $is_student"
echo "First fruit: ${fruits[0]}"

In this example, we declare variables of different types and assign values to them. Bash doesn't require explicit type declarations, but it's important to understand how different types behave.

3. Bash Commands and Arguments

Bash provides a wide range of built-in and external commands that can be used to perform various tasks. Arguments can be passed to these commands to modify their behavior.

echo "The current directory is $(pwd)"
ls -l *.txt
grep "error" log.txt

These examples demonstrate the use of the `echo`, `ls`, and `grep` commands, with various arguments and options. Command substitution is also shown using `$(pwd)`.

4. Bash Control Flow

Bash provides control flow statements, such as `if-then-else`, `case`, `for`, `while`, and `until`, to control the execution of your scripts.

if [ "$age" -lt 18 ]; then
    echo "You are a minor"
elif [ "$age" -eq 18 ]; then
    echo "You just became an adult"
else
    echo "You are an adult"
fi

for i in {1..5}; do
    echo "$i"
done

while [ "$counter" -lt 5 ]; do
    echo "Counter: $counter"
    counter=$((counter + 1))
done

These examples demonstrate the use of `if-then-else` statements, `for` loops, and `while` loops in Bash scripts.

5. Bash Functions

Bash allows you to define and call functions, which are reusable blocks of code that perform specific tasks.

greet() {
    local name=$1
    echo "Hello, $name!"
}

add() {
    local a=$1
    local b=$2
    echo $((a + b))
}

greet "Alice"
result=$(add 5 3)
echo "The result is: $result"

This example demonstrates the creation and usage of functions in Bash, including passing arguments and returning values.

6. Bash File Operations

Bash provides a set of commands and techniques for working with files and directories, such as creating, deleting, and modifying files and directories.

mkdir my_directory
touch my_file.txt
echo "Hello, file!" > my_file.txt
cat my_file.txt
cp my_file.txt backup.txt
mv my_file.txt new_name.txt
rm new_name.txt

These examples demonstrate various file operations in Bash, including creating directories, creating and modifying files, and moving or deleting files.

7. Bash Input and Output

Bash provides ways to handle input and output, including reading user input, writing to the console, and redirecting input and output streams.

read -p "Enter your name: " name
echo "Hello, $name!"

echo "This is stdout" > output.txt
echo "This is stderr" >&2

cat < input.txt
ls -l | grep ".txt"

These examples demonstrate how to use the `read` command to get user input, redirect output to files, read from files, and use pipes to connect commands.

8. Bash Scripting

Bash scripts are text files that contain a series of Bash commands and can be executed as programs. Bash scripting allows you to automate repetitive tasks and create custom tools.

#!/bin/bash

# This script demonstrates various Bash features

echo "Welcome to the Bash script demo!"

# Variables
name="Alice"
age=30

# Control flow
if [ "$age" -ge 18 ]; then
    echo "$name is an adult."
else
    echo "$name is a minor."
fi

# Loops
for i in {1..5}; do
    echo "Iteration $i"
done

# Functions
greet() {
    echo "Hello, $1!"
}

greet "Bob"

# File operations
echo "Creating a file..."
echo "This is a test file." > test.txt
cat test.txt
rm test.txt

echo "Script completed successfully!"

This example demonstrates a more complex Bash script that combines various features we've learned, including variables, control flow, loops, functions, and file operations.

9. Advanced Bash Topics

Bash has many advanced features and capabilities that you can explore further, including:

  • Associative arrays
  • Regular expressions
  • Process substitution
  • Subshells
  • Traps and signal handling
  • Debugging techniques
#!/bin/bash

# Associative array
declare -A person
person[name]="Alice"
person[age]=30
person[city]="New York"

echo "Name: ${person[name]}"

# Regular expression
if [[ "example@email.com" =~ ^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}$ ]]; then
    echo "Valid email address"
fi

# Process substitution
diff <(ls dir1) <(ls dir2)

# Subshell
(cd /tmp && ls -l)

# Trap
trap 'echo "Ctrl+C pressed"; exit 1' INT

# Debugging
set -x  # Enable debugging
echo "This line will be printed with debug info"
set +x  # Disable debugging

These examples demonstrate some advanced Bash features. As you become more comfortable with Bash, exploring these topics will help you write more powerful and efficient scripts.
