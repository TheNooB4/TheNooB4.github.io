1. Introduction to Bash

Bash (Bourne-Again SHell) is a Unix shell and command language that is widely used for scripting and automating tasks on Unix-based systems, including Linux and macOS.

#!/bin/bash
echo "Hello, World!"

This simple "Hello, World!" Bash script demonstrates the basic structure of a Bash script, including the shebang line and the `echo` command.

2. Bash Variables and Data Types

Bash supports several data types, including strings, integers, and arrays. Variables are used to store and manipulate data in Bash scripts.

name="Alice"
age=25
is_student=true

In this example, we declare variables of different types and assign values to them.

3. Bash Commands and Arguments

Bash provides a wide range of built-in and external commands that can be used to perform various tasks. Arguments can be passed to these commands to modify their behavior.

echo "The current directory is $(pwd)"
ls -l *.txt

These examples demonstrate the use of the `echo` and `ls` commands, with the `pwd` command used as a command substitution and the `-l` option passed to `ls`.

4. Bash Control Flow

Bash provides control flow statements, such as `if-then-else`, `case`, `for`, `while`, and `until`, to control the execution of your scripts.

if [ "$age" -lt 18 ]; then
    echo "You are a minor"
elif [ "$age" -eq 18 ]; then
    echo "You just became an adult"
else
    echo "You are an adult"
fi

for i in {1..5}; do
    echo "$i"
done

These examples demonstrate the use of `if-then-else` statements and `for` loops in Bash scripts.

5. Bash Functions

Bash allows you to define and call functions, which are reusable blocks of code that perform specific tasks.

add() {
    local a=$1
    local b=$2
    echo $((a + b))
}

result=$(add 5 3)
echo "The result is: $result"

This example demonstrates the creation and usage of a simple function to perform addition in a Bash script.

6. Bash File Operations

Bash provides a set of commands and techniques for working with files and directories, such as creating, deleting, and modifying files and directories.

mkdir my-directory
touch my-file.txt
echo "Hello, file!" > my-file.txt
rm my-file.txt

This example demonstrates the use of `mkdir`, `touch`, `echo`, and `rm` commands to perform various file operations in a Bash script.

7. Bash Input and Output

Bash provides ways to handle input and output, including reading user input, writing to the console, and redirecting input and output streams.

read -p "Enter your name: " name
echo "Hello, $name!"

cat file1.txt file2.txt > combined.txt

These examples demonstrate how to use the `read` command to get user input and the `echo` command to write output, as well as how to redirect input and output using the `>` operator.

8. Bash Scripting

Bash scripts are text files that contain a series of Bash commands and can be executed as programs. Bash scripting allows you to automate repetitive tasks and create custom tools.

#!/bin/bash

echo "This is a Bash script!"
echo "The current user is: $(whoami)"
echo "The current directory is: $(pwd)"

This example demonstrates the structure of a basic Bash script, including the shebang line and the use of command substitution.

9. Advanced Bash Topics

Bash has many advanced features and capabilities that you can explore further, including:

  • Associative arrays
  • Arithmetic operations
  • Regular expressions
  • Subshells and process substitution
  • Debugging and error handling
  • Bash scripting best practices
declare -A person
person[name]="Alice"
person[age]=25
person[city]="New York"

echo "Name: ${person[name]}"
echo "Age: ${person[age]}"
echo "City: ${person[city]}"

let "result = 10 + 5"
echo "The result is: $result"

These examples demonstrate the use of associative arrays and arithmetic operations in Bash scripts.
