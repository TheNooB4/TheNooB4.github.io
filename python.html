1. Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991.

Key features of Python include:

  • Easy to learn and read
  • Extensive standard library
  • Support for multiple programming paradigms
  • Dynamic typing and memory management
print("Hello, World!")

This simple "Hello, World!" program demonstrates the simplicity of Python syntax.

2. Python Basics

Let's start with the fundamental concepts of Python programming.

Variables and Data Types

Python has several built-in data types:

# Integers
age = 25

# Floating-point numbers
pi = 3.14159

# Strings
name = "Alice"

# Booleans
is_student = True

# Lists
fruits = ["apple", "banana", "cherry"]

# Dictionaries
person = {"name": "Bob", "age": 30}

# Print variables
print(f"Age: {age}")
print(f"Pi: {pi}")
print(f"Name: {name}")
print(f"Is student? {is_student}")
print(f"Fruits: {fruits}")
print(f"Person: {person}")

Basic Operators

Python supports various operators for arithmetic, comparison, and logical operations.

# Arithmetic operators
a = 10
b = 3

print(f"Addition: {a + b}")
print(f"Subtraction: {a - b}")
print(f"Multiplication: {a * b}")
print(f"Division: {a / b}")
print(f"Floor Division: {a // b}")
print(f"Modulus: {a % b}")
print(f"Exponentiation: {a ** b}")

# Comparison operators
print(f"Equal: {a == b}")
print(f"Not Equal: {a != b}")
print(f"Greater Than: {a > b}")
print(f"Less Than: {a < b}")

# Logical operators
x = True
y = False

print(f"AND: {x and y}")
print(f"OR: {x or y}")
print(f"NOT: {not x}")

3. Control Flow

Control flow statements allow you to control the execution of your code based on certain conditions.

If-Else Statements

age = 18

if age < 18:
    print("You are a minor")
elif age == 18:
    print("You just became an adult")
else:
    print("You are an adult")

For Loops

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(f"I like {fruit}")

# Range-based for loop
for i in range(5):
    print(i)

While Loops

count = 0

while count < 5:
    print(f"Count is {count}")
    count += 1

4. Functions

Functions are reusable blocks of code that perform specific tasks.

def greet(name):
    return f"Hello, {name}!"

# Call the function
message = greet("Alice")
print(message)

# Function with default parameter
def power(base, exponent=2):
    return base ** exponent

print(power(3))    # 3^2 = 9
print(power(3, 3)) # 3^3 = 27

# Lambda functions
square = lambda x: x ** 2
print(square(4))  # 16

5. Data Structures

Python has several built-in data structures that are commonly used.

Lists

fruits = ["apple", "banana", "cherry"]

# Accessing elements
print(fruits[0])  # apple

# Adding elements
fruits.append("date")
fruits.insert(1, "blueberry")

# Removing elements
fruits.remove("banana")
popped = fruits.pop()

# Slicing
print(fruits[1:3])

# List comprehension
squares = [x**2 for x in range(5)]

Dictionaries

person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}

# Accessing values
print(person["name"])

# Adding/updating key-value pairs
person["job"] = "Engineer"
person["age"] = 31

# Removing key-value pairs
del person["city"]

# Dictionary comprehension
squares = {x: x**2 for x in range(5)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Sets

fruits = {"apple", "banana", "cherry"}

# Adding elements
fruits.add("date")

# Removing elements
fruits.remove("banana")

# Set operations
other_fruits = {"cherry", "date", "elderberry"}
print(fruits.union(other_fruits))
print(fruits.intersection(other_fruits))
print(fruits.difference(other_fruits))

6. Object-Oriented Programming

Python supports object-oriented programming (OOP) paradigm.

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        return f"{self.name} says Woof!"

# Creating an instance
my_dog = Dog("Buddy", 5)
print(my_dog.bark())  # Buddy says Woof!

# Inheritance
class GuideDog(Dog):
    def __init__(self, name, age, is_guide):
        super().__init__(name, age)
        self.is_guide = is_guide

    def guide(self):
        return f"{self.name} is guiding its owner."

guide_dog = GuideDog("Bella", 7, True)
print(guide_dog.bark())      # Bella says Woof!
print(guide_dog.guide())    # Bella is guiding its owner.

In this example, we define a `Dog` class with an `__init__` method to initialize the `name` and `age` attributes, and a `bark` method to make the dog bark. We then create an instance of the `Dog` class and call the `bark` method.

We also demonstrate inheritance by creating a `GuideDog` class that inherits from the `Dog` class and adds an `is_guide` attribute and a `guide` method.

7. Modules and Packages

Python allows you to organize your code into modules and packages for better modularity and reusability.

# math_utils.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

# main.py
import math_utils

result = math_utils.add(5, 3)
print(result)  # 8

result = math_utils.subtract(10, 4)
print(result)  # 6

# Alternatively, you can import specific functions
from math_utils import add, subtract

result = add(5, 3)
print(result)  # 8

result = subtract(10, 4)
print(result)  # 6

In this example, we create a `math_utils.py` module with two functions, `add` and `subtract`. In the `main.py` file, we import the entire module and use the functions. We also demonstrate an alternative way to import specific functions from the module.

8. File Handling

Python provides built-in functions to read from and write to files.

# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, World!")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)  # Hello, World!

# Appending to a file
with open("example.txt", "a") as file:
    file.write("\nThis is a new line.")

In this example, we use the `open` function to open a file in different modes (`"w"` for writing, `"r"` for reading, and `"a"` for appending). The `with` statement ensures that the file is properly closed after the operations are completed.

9. Exception Handling

Python provides a way to handle unexpected situations or errors that may occur during program execution.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Error: Division by zero")
else:
    print(f"Result: {result}")
finally:
    print("This block will always execute")

In this example, we attempt to divide 10 by 0, which will raise a `ZeroDivisionError`. We catch the exception using the `except` block and print an error message. The `else` block will only execute if no exceptions are raised, and the `finally` block will always execute, regardless of whether an exception occurred or not.

10. Advanced Topics

Python has many advanced features and libraries that you can explore further.

List Comprehensions

List comprehensions provide a concise way to create lists.

numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
print(squares)  # [1, 4, 9, 16, 25]

Generators

Generators are a way to create iterators without using a class.

def count_up_to(n):
    i = 0
    while i < n:
        yield i
        i += 1

counter = count_up_to(5)
print(next(counter))  # 0
print(next(counter))  # 1
print(next(counter))  # 2
print(next(counter))  # 3
print(next(counter))  # 4
print(next(counter))  # StopIteration exception

Decorators

Decorators are a way to modify the behavior of a function.

def uppercase(func):
    def wrapper():
        result = func()
        return result.upper()
    return wrapper

@uppercase
def say_hello():
    return "hello"

print(say_hello())  # HELLO
