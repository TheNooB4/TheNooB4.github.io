1. Introduction to JavaScript

JavaScript is a high-level, dynamic, and interpreted programming language. It is primarily used for web development, creating interactive web pages, and building server-side applications.

console.log("Hello, World!");

This simple "Hello, World!" program demonstrates the basic syntax of JavaScript.

2. Variables and Data Types

JavaScript has several data types, including numbers, strings, booleans, objects, and more. Variables are used to store data in JavaScript.

let age = 25;
const pi = 3.14159;
let name = "Alice";
let isStudent = true;

In this example, we declare variables of different data types and assign values to them.

3. Operators and Expressions

JavaScript provides various operators, including arithmetic, assignment, comparison, and logical operators, which can be used to create expressions.

let a = 10, b = 3;
console.log("Addition: " + (a + b));
console.log("Subtraction: " + (a - b));
console.log("Multiplication: " + (a * b));
console.log("Division: " + (a / b));
console.log("Modulus: " + (a % b));
console.log("Equal: " + (a == b));
console.log("Not Equal: " + (a != b));

This code demonstrates the use of various operators to perform arithmetic and comparison operations.

4. Control Flow Statements

JavaScript provides control flow statements, such as `if-else`, `switch`, `for`, `while`, and `do-while`, to control the execution of your code.

let age = 18;
if (age < 18) {
    console.log("You are a minor");
} else if (age == 18) {
    console.log("You just became an adult");
} else {
    console.log("You are an adult");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

These examples demonstrate the use of `if-else` statements and `for` loops in JavaScript.

5. Functions

Functions in JavaScript are reusable blocks of code that perform specific tasks. They can take parameters and return values.

function add(a, b) {
    return a + b;
}

let result = add(5, 3);
console.log(result); // Output: 8

const myFunction = (x, y) => {
    return x * y;
};

result = myFunction(10, 4);
console.log(result); // Output: 40

In this example, we define two functions, one using the traditional function syntax and one using arrow function syntax, to perform addition and multiplication operations.

6. Arrays

Arrays in JavaScript are used to store collections of data.

let numbers = [1, 2, 3, 4, 5];
console.log(numbers[0]); // Output: 1

let fruits = ["apple", "banana", "cherry"];
fruits.push("orange");
console.log(fruits); // Output: ["apple", "banana", "cherry", "orange"]

These examples demonstrate how to create and manipulate arrays in JavaScript.

7. Objects

Objects in JavaScript are collections of key-value pairs, similar to dictionaries or hash tables.

let person = {
    name: "Alice",
    age: 25,
    isStudent: true
};

console.log(person.name); // Output: "Alice"
person.city = "New York";
console.log(person); // Output: { name: "Alice", age: 25, isStudent: true, city: "New York" }

This example demonstrates how to create and access properties of a JavaScript object.

8. DOM Manipulation

JavaScript provides the ability to interact with the Document Object Model (DOM), allowing you to dynamically modify the structure and content of web pages.

const heading = document.getElementById("myHeading");
heading.textContent = "Welcome to my website!";

const paragraphs = document.getElementsByTagName("p");
for (let i = 0; i < paragraphs.length; i++) {
    paragraphs[i].style.color = "blue";
}

const button = document.querySelector("button");
button.addEventListener("click", () => {
    alert("Button clicked!");
});

This example demonstrates how to select and manipulate HTML elements using JavaScript's DOM API.

9. Events

JavaScript allows you to handle various events, such as clicks, key presses, and form submissions, to create interactive user experiences.

const button = document.querySelector("button");
button.addEventListener("click", () => {
    alert("Button clicked!");
});

const input = document.getElementById("myInput");
input.addEventListener("keyup", (event) => {
    console.log("Key pressed: " + event.key);
});

This example demonstrates how to add event listeners to HTML elements and handle click and keyup events.

10. Advanced JavaScript Topics

JavaScript has many advanced features and concepts that you can explore further, including:

  • Asynchronous JavaScript (Promises, Async/Await)
  • ES6 and beyond (let, const, arrow functions, classes)
  • Module system (CommonJS, ES6 Modules)
  • Browser APIs (Fetch, Web Storage, Web Workers)
  • JavaScript Frameworks and Libraries (React, Angular, Vue.js)
  • Node.js and server-side JavaScript
// Asynchronous JavaScript
const fetchData = () => {
    return new Promise((resolve, reject) => {
        // Simulating an asynchronous operation
        setTimeout(() => {
            resolve("Data fetched successfully!");
        }, 2000);
    });
};

fetchData()
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error(error);
    });

// ES6 Features
const PI = 3.14159;
let radius = 5;
let area = PI * radius ** 2;
console.log(`The area of a circle with radius ${radius} is ${area}.`);

These examples demonstrate the use of asynchronous JavaScript with Promises and the new features introduced in ES6 (and beyond).
